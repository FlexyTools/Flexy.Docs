# Start Guide

## First steps
- Install Flexy.AssetRefs, Flexy.Core, Flexy.GameFlow from AssetStore or github to your project
- import Flexy.Game! unitypackage from GameFlow/Content directory

## Structure
- !Game - directory with all setup 
  - !Code - game scripts lie
  - Flow - place for flow bootstrap and states prefabs
  - Boot.scene - ready to use boot scene
