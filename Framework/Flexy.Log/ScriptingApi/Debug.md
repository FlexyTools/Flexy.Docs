# Debug

[Flexy.Tools](../../../Readme.md) / [Framework](../../Readme.md) / [Flexy.Log](../Readme.md) / [Scripting Api](Readme.md) / Debug


## Description


Main class to use for debug logs  
It mimics UnityEngine.Debug class because it designed to be drop in replacement for it  
In addition, it has 2 extension points  

## Properties

| Property                         | Description                                                                                                                    |  
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| All UnityEngine.Debug properties | Read description of them in [official](https://docs.unity3d.com/6000.3/Documentation/ScriptReference/Debug.html) documentation |
|                                  |                                                                                                                                |
| Filter?                          | Log Filter implementation (default is LogFilterPerMember)                                                                      |
| FormatLogStringMethod            | Set Method for custom log format                                                                                               |
