# Misc

Different uncategorized useful classes


| Class                    | Description                                                                                                                                                                                                                                 |  
|:-------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| UIWindow                 | It is empty implementor of Flexy.GameFlow.State just to explicitly show that it is a window                                                                                                                                                 |
| UIGradient               | UIMeshEffect to add 4 corders gradient to ui image                                                                                                                                                                                          |  
| EmptyGraphic             | Implement Graphic without any View. Useful for invisible ui Blockers or Button clickable area extenders                                                                                                                                     |
| AutoCanvasScaler         | Enhances CanvasScaler with auto rotate reference resolution in Expand mode                                                                                                                                                                  |
| BindableAppInfo          | Ready to use Bindable behaviour to bind to Flexy.AssetRefs AppInfo and Unity Application classes                                                                                                                                            |
| EarlyInputAndEventSystem | Enhanced EventSystem that moves Unity InputSystem inputs read to EarlyUpdate followed by UI SventSystem Update to process input <br> So FixedUpdate and Update gameplay code can read full input from devices and UI without frame or delay |
