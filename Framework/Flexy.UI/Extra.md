# Extra

Extra classes to speed up development

| Class                  | Description                                                                              |  
|:-----------------------|:-----------------------------------------------------------------------------------------|
| ButtonClick_OpenWindow | Helper Class to create window in ButtonClick in prototypes (dont use in production code) |
| Window_Empty           | Ready to use Empty Window for windows with static content in prototypes                  |  
| Window_PlayVideo       | Ready to use Window to play video through VideoPlayer (usage sample)                     |
| Window_ShowImage       | Ready to use window to show static image for some time (useful for splash screens)       |
